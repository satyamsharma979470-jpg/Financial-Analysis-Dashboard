# Customer Churn Analysis - Power BI Project

## 📌 Project Overview
This project analyzes customer churn behavior in a European bank using Power BI. The goal is to identify patterns and factors that lead to customer attrition.

## 🎯 Objectives
- Identify churn patterns
- Analyze customer behavior
- Provide business insights for retention strategies

## 📊 Tools Used
- Power BI
- DAX
- Data Visualization

## 📁 Dataset Features
- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Active Member Status
- Estimated Salary
- Churn (Exited)

## 📈 Key Insights
- Customers with low engagement are more likely to churn
- Germany shows higher churn rate
- Customers with fewer products tend to leave more
- Active members have significantly lower churn

## 📌 Dashboard Features
- KPI Metrics (Churn Rate, Total Customers)
- Geography-wise analysis
- Age group analysis
- Product usage trends
- Interactive filters

## 🚀 Outcome
This project helps banks proactively identify high-risk customers and design better retention strategies.
